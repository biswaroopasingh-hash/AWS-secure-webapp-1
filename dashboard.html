Request Flow — Step by Step
1
User opens login page (index.html via Amplify) AWS WAF inspects the request first. Clean traffic reaches the Amplify-hosted login page.
2
User enters email + password → Cognito authenticates Login page calls Cognito InitiateAuth API directly. Cognito validates credentials and returns JWT tokens.
3
JWT tokens saved → Dashboard opens IdToken and AccessToken stored in sessionStorage. User is redirected to dashboard.html automatically.
4
Dashboard verifies token on load dashboard.html checks sessionStorage on every load. No token = immediate redirect back to login. Valid token = dashboard renders.
5
Data sent to API Gateway with JWT header POST /submit called with Authorization: Bearer <idToken>. API Gateway Cognito Authorizer validates the token.
6
Lambda saves to DynamoDB → CloudWatch logs LoginFormHandler writes email + name to LoginData table. Every step is automatically logged to CloudWatch.
